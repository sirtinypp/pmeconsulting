# Content Submission Guide: phmedtecheu G-LMS
**Step-by-Step Instructions for Providing Curriculum Data**

**Objective**: To ensure the **phmedtecheu** curriculum (A1–C1) is formatted correctly for rapid system upload.

---

## 1. Lesson Tracks & Categories
For each course (A1, B1, etc.), please provide:
*   **Track Title**: (e.g., "A1: Hospital Basics").
*   **Categories**: (e.g., "Medical Vocabulary," "Speaking Foundations," "Grammar").
*   **Prerequisites**: Which lessons must be completed before this one?

---

## 2. Resource Formats
The system supports three primary resource types. Please provide them as follows:

### **A. Video Lessons**
*   **Format**: YouTube (Unlisted) or Vimeo links.
*   **Metadata**: Title, Duration (min), and a brief description (2–3 sentences).

### **B. PDF & Document Resources**
*   **Format**: PDF only (Standardized for mobile viewing).
*   **Metadata**: Resource Name (e.g., "Clinical Vocabulary Checklist").

### **C. Reading Materials**
*   **Format**: Plain text or Markdown (I can convert Word docs if needed).

---

## 3. Quiz & Exercise Structure
To populate the **Quiz Engine**, for each level, please provide a list of questions in the following format:

**Question**: [The actual question text, e.g., "What is the German word for 'Nurse'?"]  
**Choices**:  
1. [Correct Answer] - Krankenschwester  
2. [Distractor] - Arzt  
3. [Distractor] - Apotheker  
4. [Distractor] - Patient  
**Explanation**: [Optional - Shown after the student answers] - *Krankenschwester* translates literally to 'Hospital Sister'.

---

## 4. Speaking Sessions & CV Booking
For the premium services, please provide:
*   **Availability Slots**: (How many sessions per week/month are included in the 'Standard' vs. 'Premium' tiers?)
*   **Booking Instructions**: Where should students be directed to book? (Calendly link or internal system?)

---

## 5. Submission Method
Please upload all curriculum files to the shared **Google Drive / Dropbox** folder in organized sub-folders:
*   `A1_Foundations/`
    *   `Vocabulary/`
    *   `Videos/`
*   `A2_Clinical/`
*   `B1_Professional/`

**Note**: For batch uploads, a **CSV or Excel** spreadsheet is preferred for the quiz questions. I can provide a template.
